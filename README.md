
🚀 CareerLens
✨ Confused about which career suits your skills?
CareerLens is an intelligent career guidance platform that helps users explore career paths based on their skills and interests. Simply select your skills, and CareerLens provides personalized career recommendations powered by AI.

🎯 Features
🔹 Skill-based career recommendations
🔹 Machine learning-powered predictions
🔹 Interactive Streamlit interface
🔹 Personalized career insights
🔹 Clean and beginner-friendly design

🛠️ Tech Stack
🐍 Python
🤖 Scikit-learn
📊 Pandas & NumPy
🎨 Streamlit
